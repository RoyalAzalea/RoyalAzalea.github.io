python과 ruby에서 다루는 문자열 관련된 기능들은 매우 비슷하다.
python에서 ""과 ''는 거의 같은 의미이다. ruby에서는 살짝 다르다.
간단하게 문자열을 출력해보겠다.

**python**

```python
print('Hello')
print("Hello")
print("Hello 'world'")
print('Hello "world"')
```

**ruby**

{% highlight ruby %}
puts('Hello')
puts("Hello")
puts("Hello 'world'")
puts('Hello "world"')
{% endhighlight %}

**result**
---
Hello
Hello
Hello 'world'
Hello "world"
---

다음은 문자열
